## 任务描述

实现工程驾驶舱运营 Tab 的 P1 静态 Mock 版本（`GrowthTab`）。使用 `IS_MOCK = true` 常量控制数据来源，渲染三个面板：DAU/MAU 趋势、用户留存/漏斗、用户反馈聚合。

P1 阶段全部使用 Mock 数据，不接入任何真实数据平台（Amplitude / Mixpanel / PostHog 等）。P2 阶段通过 Feature Flag 切换，另立 SDD。

### 包含

- `IS_MOCK` 常量定义（`const IS_MOCK = true`），置于文件顶部
- 三个 Mock 数据面板渲染：
  1. DAU/MAU 趋势（Growth Metrics）：模拟 7 天 DAU 数据表格/列表
  2. 用户留存/漏斗（Retention）：模拟留存率数据（Day1/Day7/Day30）
  3. 用户反馈聚合（Feedback）：模拟 3~5 条用户反馈，含渠道/时间/内容
- 将 cockpit.tsx 运营 Tab 占位替换为 `<GrowthTab />`
- Mock 数据标注"数据仅供演示"提示

### 不包含（由其他 TASK 负责）

- 真实数据平台 API 接入（P2，另立 SDD）
- A/B Test 结果摘要（P2）
- 关键事件排行（P2）
- CockpitPage 布局框架（由 TASK-002-03 负责）
