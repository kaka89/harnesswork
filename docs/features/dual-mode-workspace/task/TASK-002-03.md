## 任务描述

在 TASK-002-01 建立的 CockpitPage 骨架基础上，实现完整的工程驾驶舱容器页，包含上下分栏布局、`createSignal` 管理 Tab 局部状态、顶部"返回模式选择"入口，以及独立的 `CockpitTabNav` 无状态导航组件（支持键盘导航、当前 Tab 高亮）。

子 Tab 内容区在本 TASK 中只渲染占位符，实际内容由 TASK-002-04/05/07/08 分别实现。

### 包含

- CockpitPage 完整布局：顶部固定导航栏（含返回入口 + CockpitTabNav）+ 下方内容区
- `createSignal<TabId>` 管理 activeTab 局部状态，默认值 `"product"`
- CockpitTabNav 无状态组件：四 Tab 渲染、active 样式、键盘导航（Arrow / Tab / Enter / Space）
- 顶部"返回模式选择"按钮：点击 navigate `"/mode-select"`
- Tab 切换 < 100ms（Signal 驱动，无网络请求）
- 四个 Tab 内容区占位符（`<div data-testid="tab-{name}-placeholder">`)

### 不包含（由其他 TASK 负责）

- ProductTab 实际内容（由 TASK-002-04 负责）
- EngineeringTab 实际内容（由 TASK-002-05 负责）
- ReleaseTab 实际内容（由 TASK-002-07 负责）
- GrowthTab 实际内容（由 TASK-002-08 负责）

## 实现要点

- [ ] `apps/app/src/app/pages/cockpit.tsx` — CockpitPage 完整实现
  - 保留 `TabId` 类型导出（来自 TASK-002-01）
  - 使用 `createSignal<TabId>("product")` 声明 `[activeTab, setActiveTab]`
  - 整体布局：
    ```tsx
    <div class="flex flex-col h-screen bg-gray-950 text-white" data-testid="cockpit-page">
      {/* 顶部固定栏：返回入口 + Tab 导航 */}
      <header class="flex items-center border-b border-gray-800 px-4 py-2 shrink-0">
        <button onClick={() => navigate("/mode-select")} ...>← 返回模式选择</button>
        <CockpitTabNav activeTab={activeTab()} onTabChange={setActiveTab} />
      </header>
      {/* 内容区：撑满剩余高度 */}
      <main class="flex-1 overflow-hidden">
        {/* Switch by activeTab() */}
      </main>
    </div>
    ```
  - 内容区用 SolidJS `<Switch>` / `<Match>` 按 activeTab 渲染对应占位 div
  - `data-testid="cockpit-page"` 在根容器保留

- [ ] `apps/app/src/app/components/cockpit/tab-nav.tsx` — CockpitTabNav 实现
  - Props 接口（复用 MODULE-002 定义）：
    ```typescript
    interface CockpitTabNavProps {
      activeTab: TabId;
      onTabChange: (tab: TabId) => void;
    }
    ```
  - 渲染四个 Tab 按钮，label 与 id 映射：
    - `product` → "产品"
    - `engineering` → "研发"
    - `release` → "发布&运维"
    - `growth` → "运营"
  - Active 样式：`border-b-2 border-blue-500 text-blue-400`；默认样式：`text-gray-400 hover:text-gray-200`
  - 键盘导航（BH-15）：
    - `onKeyDown` 处理 `ArrowLeft` / `ArrowRight`：焦点在四 Tab 间循环移动
    - `Enter` 或 `Space`：触发 `onTabChange(currentFocusedTab)`
    - 各 Tab 按钮设置 `tabIndex={activeTab === tab ? 0 : -1}`（roving tabindex 模式）
  - `data-testid={`tab-${tab}`}` 绑定在每个 Tab 按钮

- [ ] 内容区占位 Tab 组件（临时，后续由各 TASK 替换）
  - 在 cockpit.tsx 内部定义四个 inline 占位函数组件或直接返回 `<div>`
  - 命名如：`ProductTabPlaceholder`、`EngineeringTabPlaceholder` 等
  - 每个占位 div 带 `data-testid="tab-product-placeholder"` 等属性

- [ ] navigate 依赖
  - `const navigate = useNavigate()` 来自 `@solidjs/router`，在 CockpitPage 函数体内调用
