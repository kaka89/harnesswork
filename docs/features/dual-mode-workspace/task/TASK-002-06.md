## 任务描述

在 `apps/server` 中实现两个新 HTTP 端点：`GET /docs`（列举工作区 `docs/` 目录树）和 `GET /docs/:path`（读取指定 Markdown 文件内容）。

两个端点是产品 Tab（TASK-002-04）的服务端依赖，必须在 TASK-002-04 联调前完成。本 TASK 与前端 TASK-002-03 可并行执行，无代码冲突。

### 包含

- GET /docs：遍历工作区 `docs/` 子目录，提取 .md 文件，解析 frontmatter，返回 `DocEntry[]`
- GET /docs/:path：读取指定路径 Markdown 文件原始内容，路径穿越防护（403 FORBIDDEN）
- 深度限制：仅收录 `docs/` 下相对路径深度 ≤ 4 层的文件（BH-05）
- frontmatter 解析：提取 `meta.title` 和 `meta.status`；缺失时回退为文件名 / `"unknown"`（BH-06）
- 路由注册：在 server 入口文件中注册 `/docs` 路由模块

### 不包含（由其他 TASK 负责）

- 前端 DocTreePanel / DocViewerPanel 调用（由 TASK-002-04 负责）
- OpenAPI 文件修改（MODULE-002 的 openapi/cockpit-docs-v1.yaml 已是规格来源，本 TASK 按其实现）

## 实现要点

- [ ] 确认 `apps/server` 框架（Hono / Express / Fastify / 其他）
  - 查看 `apps/server/src/index.ts`（或 `app.ts`）确认路由注册方式

- [ ] 创建 `apps/server/src/routes/docs.ts` — 文档路由模块
  - **GET /docs 端点**：
    ```typescript
    // 获取工作区根路径（server 启动时注入或环境变量）
    const WORKSPACE_ROOT = process.env.WORKSPACE_ROOT ?? process.cwd();
    const DOCS_ROOT = path.join(WORKSPACE_ROOT, "docs");

    app.get("/docs", async (c) => {
      if (!fs.existsSync(DOCS_ROOT)) return c.json({ error: "NOT_FOUND" }, 404);
      const entries = await collectDocEntries(DOCS_ROOT, 0, 4);
      return c.json(entries);
    });
    ```
  - `collectDocEntries(dir, depth, maxDepth)` 递归收集 `.md` 文件：
    - `depth > maxDepth` 时停止（BH-05：深度 ≤ 4）
    - 对每个 `.md` 文件调用 `parseDocEntry(filePath)` 提取字段
    - 返回路径使用相对于 `docs/` 的路径，分隔符统一用 `/`
  - `parseDocEntry(filePath)` 解析单个文件：
    - 读取文件内容，检测 `---` 包裹的 YAML frontmatter
    - 提取 `meta.title` → `title`，`meta.status` → `status`
    - 无 frontmatter 或缺失字段时回退（BH-06）：`title = fileName`，`status = "unknown"`
    - `type` 字段：优先 `meta.type`，其次路径前缀推断（`product/prd` → `prd`，`product/architecture` → `sdd` 等）
  - **GET /docs/:path 端点**：
    ```typescript
    app.get("/docs/:path{.+}", async (c) => {
      const rawPath = c.req.param("path");
      const decodedPath = decodeURIComponent(rawPath);          // URL 解码（BH-12）
      const resolved = path.resolve(DOCS_ROOT, decodedPath);
      // 路径穿越防护（BH-10）
      if (!resolved.startsWith(DOCS_ROOT + path.sep) && resolved !== DOCS_ROOT) {
        return c.json({ error: "FORBIDDEN" }, 403);
      }
      if (!fs.existsSync(resolved)) return c.json({ error: "NOT_FOUND" }, 404);
      try {
        const content = await fs.promises.readFile(resolved, "utf-8");
        return c.text(content, 200, { "Content-Type": "text/markdown; charset=utf-8" });
      } catch {
        return c.json({ error: "INTERNAL_ERROR" }, 500);
      }
    });
    ```
