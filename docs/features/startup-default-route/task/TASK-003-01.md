## 实现要点

### 文件 1：`apps/app/src/app/entry.tsx`

**删除旧逻辑：**
```typescript
// 删除以下内容
const pref = platform.storage("harnesswork").getItem("mode-preference");
if (pref === "cockpit" && location.pathname === "/") {
  navigate("/cockpit");
}
```

**替换为新逻辑：**
```typescript
onMount(() => {
  // 冷启动检测：sessionStorage 无标记时重定向到模式选择页
  if (location.pathname === "/" && !sessionStorage.getItem("harnesswork:started")) {
    sessionStorage.setItem("harnesswork:started", "1");
    navigate("/mode-select");
  }
});
```

**注意**：
- `platform.storage` 相关的 import 和对象引用可完整保留（其他地方可能用到）
- 如果 `platform.storage` 仅用于旧逻辑，可移除对 `usePlatform` 的引用（检查是否有其他使用）
- `sessionStorage` 直接访问原生 API，无需通过 `platform.storage`

### 文件 2：`apps/app/src/app/pages/mode-select.tsx`

**新增 preference 信号（在组件函数顶部）：**
```typescript
import { createSignal } from "solid-js";

// 在组件内顶部
const [preference, setPreference] = createSignal<string | null>(
  storage.getItem("mode-preference")
);
```

**修改 openwork 卡片 button 的 class：**
```tsx
class={`flex-1 flex flex-col gap-3 p-6 rounded-xl border cursor-pointer bg-gray-900 transition-all text-left
  ${preference() === "openwork"
    ? "border-gray-400 ring-2 ring-gray-300/60 hover:border-gray-300 hover:bg-gray-800"
    : "border-gray-700 hover:border-gray-400 hover:bg-gray-800"
  }`}
```

**修改 cockpit 卡片 button 的 class：**
```tsx
class={`flex-1 flex flex-col gap-3 p-6 rounded-xl border cursor-pointer bg-gray-900 transition-all text-left
  ${preference() === "cockpit"
    ? "border-blue-400 ring-2 ring-blue-400/60 hover:border-blue-300 hover:bg-blue-950"
    : "border-blue-600 hover:border-blue-400 hover:bg-blue-950"
  }`}
```

**handleSelectOpenwork 中同步更新 preference signal：**
```typescript
const handleSelectOpenwork = () => {
  storage.setItem("mode-preference", "openwork");
  setPreference("openwork");     // 保持 signal 同步（虽然会立即 navigate，防止闪烁）
  navigate("/");
};
```

**handleSelectCockpit 中同步更新 preference signal：**
```typescript
const handleSelectCockpit = () => {
  storage.setItem("mode-preference", "cockpit");
  setPreference("cockpit");
  navigate("/cockpit");
};
```

## 测试要求

| 测试场景 | 验证点 |
|---------|--------|
| BH-01：无 preference，无 started 标记 | navigate 调用 `/mode-select`；两卡片均无 `ring-2` class |
| BH-02：preference=cockpit，无 started 标记 | navigate 调用 `/mode-select`；cockpit 卡片有 `ring-2` |

2. 点击 cockpit → 刷新 → 确认跳转 `/mode-select`，cockpit 卡片高亮
3. 点击 openwork（进入 `/`）→ 在 openwork 页不触发二次重定向
4. 点击 openwork → 刷新 → 确认跳转 `/mode-select`，openwork 卡片高亮
